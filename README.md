# Peluchilandia

> Sitio web E-commerce de peluches - DAW (Diseño de Aplicaciones Web)

## [Página web](https://peluchilandia.angelzhang.dev)

## Descripción

Peluchilandia es una tienda en línea especializada en la venta de peluches. Este proyecto forma parte del módulo de Diseño de Interfaces Web y presenta una experiencia de compra completa con catálogo de productos, carrito de compras y proceso de checkout.

## Características

- Página de inicio con catálogo de productos
- Vista detallada de productos individuales
- Carrito de compras funcional
- Proceso de compra y checkout
- Diseño responsive
- Interfaz moderna y amigable
- Página "Sobre Nosotros"

## Estructura del Proyecto

```
peluchilandia/
├── index.html              # Página principal
├── producto.html           # Vista de producto individual
├── carrito.html            # Carrito de compras
├── compra.html             # Proceso de checkout
├── sobre-nosotros.html     # Información de la empresa
├── style.css               # Estilos globales
├── index.js                # JavaScript página principal
├── producto.js             # JavaScript vista producto
├── carrito.js              # JavaScript carrito
├── producto-data.js        # Datos de productos
└── img/                    # Imágenes y recursos
```

## Instalación y Uso

1. **Clonar el repositorio**
   ```sh
   git clone https://github.com/AngelZhang159/peluchilandia.git
   cd peluchilandia
   ```

2. **Abrir en el navegador**
   - Abre el archivo `index.html` en tu navegador web
   - O utiliza un servidor local como Live Server en VS Code

3. **Navegación**
   - Explora el catálogo en la página principal
   - Haz clic en los productos para ver detalles
   - Añade productos al carrito
   - Completa el proceso de compra

- **Validación de formularios**: Verificación de datos en el proceso de compra
